
# AttendMitra

AttendMitra is a smart attendance tracker app for college students, built using Flutter. It helps students easily track their lecture attendance, provides insights and statistics, and empowers them to make informed decisions about attending or skipping classes—solving the perpetual dilemma of "KONSA CLASS BUNK KARNA HAI!"

## Features

- **Attendance Tracking**  
  Swipe through your scheduled classes for the day and mark attendance based on participation.

- **Insights Dashboard**  
  Get a clear overview of classes attended vs. classes skipped—so you never lose track of your attendance stats.

- **Weekly Analytics**  
  View attendance streaks and individual subject stats for the week and month.

- **Subject Management**  
  Add, edit, and remove subjects as per your current semester.

- **Customizable Timetable**  
  Set and modify your class timetable for each day with an easy drag-and-drop UI.

- **Decision-Assist Insights**  
  Instantly know which lectures you can afford to skip and which are critical to attend based on your attendance history.

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/<your-github-username>/AttendMitra.git
cd AttendMitra
```

2. **Install dependencies**

```bash
flutter pub get
```

3. **Run the app**

```bash
flutter run
```

## Usage

- Launch the app and set up your subjects and weekly timetable.
- Track classes live by swiping right/left to mark as attended or skipped.
- Check your weekly/monthly insights and stats to help plan your attendance and decide which lectures can be safely missed.
- Edit your timetable and subjects anytime as your semester changes.
